About D-BAS-Search
==================

**D-BAS-Search** is a microservice designed for `D-BAS <https://github.com/hhucn/dbas/>`_.

This microservice is designed to work with `Elasticsearch <https://www.elastic.co/guide/en/elasticsearch/reference/current/index.html>`_
and `Docker <https://www.docker.com/>`_. 

D-BAS-Search provides a text search for D-BAS.

The search supports:

.. toctree::
   :maxdepth: 5

   Synonyms
   Fuzzy
   Full
   Update

Check the out for more details.
